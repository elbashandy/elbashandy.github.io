Detailed Description

an algorithm that transforms NetCDF CF-2 time variable into an absolute date. Transform NetCDF CF-2 time variable into an absolute date. By default the "time" column is used, but this can be over road by set_active_column methods. the table must have temporal metadata containing base date and calendar units following the CF-2 convention. the output table will contain year,month,day,hours, minutes columns.

NOTE: this should be used in serial, as the udunits package loads an xml file in each instance. The CalCalcs package also has thread safety issues.
