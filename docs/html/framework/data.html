class teca_mesh : public teca_dataset¶

a base class for geometric data

The mesh declares containers for typical geometricly associated data such as point, cell, face and edge centered data arrays and defines the API’s for accessing them. API’s for accessing common metadata such as time related metadata are declared here.

Subclassed by teca_arakawa_c_grid, teca_cartesian_mesh, teca_curvilinear_mesh, teca_uniform_cartesian_mesh
